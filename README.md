# Netflix Page Clone

A responsive Netflix homepage clone built with HTML, CSS, and JavaScript, replicating the core look and feel of Netflix's landing/browse page.

## Features
- Netflix-style hero banner with background image
- Movie/show card grid with hover effects (scale on hover)
- Sign-in / Start buttons with custom styling
- Responsive layout

## Built With
- HTML5
- CSS3
- JavaScript
